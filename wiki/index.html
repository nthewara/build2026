Microsoft Build 2026 — LLM Wiki

A densely cross-linked knowledge base compiled from the Build 2026 session notes.

This is a Karpathy-style “LLM wiki” (the pattern Andrej Karpathy popularised): instead of reading notes linearly, knowledge is broken into concept nodes that link to the sessions they appear in and to each other, so you traverse the material by following links and the graph view. Every concept here is a topic/ tag drawn from this site’s 127 session notes — this page is purely a re-organisation of the existing content, with no new information added.

How to read it: start from a concept below or open the graph view; click a node to jump to its page; from there follow Related concepts to neighbouring ideas, or open any session to read the full notes on the main site.
